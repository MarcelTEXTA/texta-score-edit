# TEXTA score edit

**Le nouvel éditeur de partition musicale sur le web**

</div>

## Pourquoi TEXTA score edit ?

TEXTA score edit est un éditeur de partition musicale en ligne qui permet aux musiciens, compositeurs ou même enseignants de créer et modifier des partitions musicales facilement. Il offre une interface intuitive et des fonctionnalités avancées pour répondre aux besoins des utilisateurs de tous niveaux.

Vous créez des fichiers localement mais l'application s'exécute dans votre navigateur. Vous pouvez donc l'utiliser sur n'importe quel appareil sans installation préalable (pas addaptée pour les mobiles pour l'instant).

## Fonctionnalités principales

- **Édition de partitions** : Ajoutez, modifiez et supprimez des notes, des silences, des articulations et d'autres éléments musicaux.
- **Support des instruments** : Créez des partitions pour différents instruments et ajustez les portées en conséquence.
- **Exportation** : Exportez vos partitions au format PDF ou MusicXML pour les partager ou les imprimer.
- **Interface conviviale + vie privée** : Nous avons conçu une interface simple et intuitive pour que vous puissiez vous concentrer sur la création musicale sans distractions (aucune publicité, pas de compte à créer, pas de tracking).

## Utilisation

*L'application est encore en développement, mais vous pouvez déjà l'essayer en téléchargeant le code source et nous faire part de vos retours !*

### Utilisation du code

Nous utilisons [Vite](https://vitejs.dev/) pour le développement et la construction de l'application. Pour commencer, vous devez cloner le dépôt et installer les dépendances.

```bash
git clone https://github.com/MarcelTEXTA/texta-score-edit.git
cd texta-score-edit
npm install
```

### Lancer l'application en mode développement

Utilisez la commande suivante pour démarrer l'application en mode développement. Cela lancera un serveur local et ouvrira l'application dans votre navigateur par défaut. Et vous pourrez voir les modifications en temps réel lorsque vous modifiez le code source.

```bash
npm run dev
```
